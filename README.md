# Inventory-analysis-dashboard-powerbi
Inventory Analysis Dashboard using Power BI to identify stockout risks and reorder points.

## Overview

This project was developed to monitor inventory levels and identify products at risk of stockout.

## Business Problem

Inventory shortages can affect sales, customer satisfaction, and operational continuity.

The objective was to create a dashboard capable of:

- Monitoring inventory levels
- Tracking daily demand
- Calculating reorder points
- Generating purchase alerts

## Tools Used

- Power BI
- Excel
- DAX

## Key Metrics

- Current Stock
- Daily Demand
- Reorder Point
- Inventory Value
- Purchase Alerts

## Key Finding

The analysis identified a product with:

- Current Stock: 1
- Reorder Point: 2
- Daily Demand: 1

This indicates a potential stockout risk and suggests the need for replenishment.

## Business Impact

The dashboard helps:

- Reduce stock shortages
- Improve inventory visibility
- Support purchasing decisions
- Prioritize replenishment actions
